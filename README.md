# Egoeng

영어 학습을 위한 종합 플랫폼 애플리케이션입니다. 단어, 문장 정렬, 번역, 퀴즈, 학습 진도 추적 등 다양한 기능을 제공합니다.

## 📋 프로젝트 개요

- **언어**: Java 21
- **프레임워크**: Spring Boot 3.3.4
- **데이터베이스**: PostgreSQL
- **빌드 도구**: Gradle

## 🚀 주요 기능

- **인증/인가**: OAuth2, JWT 기반 보안 인증
- **단어 관리**: 단어 등록, 조회, 카테고리 관리
- **학습**: 개인화된 학습 경로 관리
- **퀴즈**: 다양한 유형의 퀴즈 시스템
- **채팅**: 실시간 채팅 기능
- **알림**: 예약된 알림 시스템
- **번역**: Papago API를 활용한 자동 번역
- **LLM 연동**: AI 기반 기능 지원
